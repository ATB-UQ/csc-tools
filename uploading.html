After datasets have been successfully exported, the archives in export_path can be uploaded to the ACSC server using the UQ RDM Cloud interface.

To upload a dataset, click on the link to your UQ RDM shared folder as described in Accessing UQ RDM Shared Folders and upload the archives created in Exporting Datasets via the UQ RDM Cloud interface.
